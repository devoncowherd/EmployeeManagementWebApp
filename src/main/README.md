# Employee Management System

---

- This web application stores employee Name, Email, Gender, and Country.


# Dependencies

---

- JDK/JRE 1.8.0
- Spring Web
- Hibernate 5.6.10
- Spring Data JPA
- Thymeleaf
  - 5.3.15
- MySQL JDBC
  - MySQL jconnector 8.0.30
- Spring Boot DevTools
- Webjars 
  - bootstrap
  - locator.core
